# Test Suite

This directory contains SystemVerilog/Verilog test cases used to validate simulator functionality.

## Credits

All files matching the pattern `pr*.v` were taken from the **Icarus Verilog (iverilog) test suite**.

Icarus Verilog is an open-source Verilog simulation and synthesis tool. The original test cases are part of the official Icarus Verilog repository and are used here for compatibility and validation testing.

Project: https://github.com/steveicarus/iverilog

## Purpose

These tests are included to:

- Validate language feature support
- Verify simulator correctness
- Ensure compatibility with established Verilog test suites
